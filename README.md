# Zebrafish-growth
data and code to analyse growth trajectories of 2n and 3n zebrafish under different temperature and oxygen conditions

Data repository content
Data files 
Datafish1-114.csv
Data file containing information about growth (length and weight) of individual fish and used for statistics and creating the figures in the manuscript. 
Abbreviations:
fish_id: a number give to identify individual zebrafish 
 r_temp: rearing temperature (23°C, 28°C)
a_temp: acclimation temperature (23°C, 28°C)
ploidy: ploidy level (2n, 3n)   
ox: oxygen level (hypoxia, normoxia)
samp_date: sampling date
sex: sex (male, female)  
weight: weight at time of sampling 
length: length 
mr_ch:  metabolic rate channel (1-8), the channel of the oxy-10 oxygen meter
tp_date: Thermal preference date, date of the thermal preference experiments
f_date: fertilization date
age: age of the fish in weeks
batch: Fertilization batch (b1, b2, b3, b4, b5, b6)
  	

Groeidata R.csv
Data file containing information about growth (length) of fish and used for statistics and creating the figures in the manuscript.
Abbreviations:


Tank: specific tank/aquarium registration number 
condition: experimental condition (AB 2n 28°C, AB 2n 23°C, AB 3n 28°C, AB 3n 23°C)
temp: rearing temperature (28°C, 23°C)
ploidy: ploidy level (2n, 3n)      
fdate: fertilization date
fish: identification number of fish (fish with same numbers are not the same individuals between measurements)
age: age in days   
tl:  total length of the fish in millimeters
cond: experimental condition (2n 28°C, 2n 23°C, 3n 28°C, 3n 23°C)
